Aby uruchomić sekcję backendową należy otworzyć terminal w folderze
 backend i zainstalować wymagane pakiety: npm i
Uruchomimć bazę danych z folderu backend/db/docker komendą: docker-compose up
Uruchomić serwer z folderu backend komendą: npm app.js
Uruchomić sekcję frontendową z folderu frointend_Vue/spacetrip komendą: npm run serve
