Jak uruchomić:
1. Należy otworzyć terminal w folderze /backend i zainstalować wymagane pakiety komendą: npm i
2. Uruchomić bazę danych z folderu /backend/db/docker komendą: docker-compose up
3. Uruchomić serwer z folderu /backend komendą: npm app.js
4. Uruchomić sekcję frontendową z folderu /frontend_Vue/spacetrip komendą: npm run serve
